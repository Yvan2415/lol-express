# be-loler

## 目标
	1. 实现用户的注册, 登录, 验证码, token, cookie, session等功能
	2. 实现数据的收集, 包括英雄数据, 技能数据, 背景故事等, 用列表菜单的形式展示

## 使用文档	
	1. 请求静态资源 /images/filename.ext  || /javascripts/filename.ext


## 依赖使用
	1. cookieParser
	2. morgan
	3. http-errors
	4. jade
	5. mongoose[https://mongoosejs.com/]
	6. debug
